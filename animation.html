Animation

Con jQuery, puede crear animaciones personalizadas.

Efectos jQuery - Animación

Animaciones jQuery - El método animate()
El animate()método jQuery se utiliza para crear animaciones personalizadas.
Sintaxis:
$(selector).animate({params},speed,callback);
El siguiente ejemplo demuestra un uso simple del animate()método; mueve un elemento

a la derecha, hasta que alcanza una propiedad izquierda de 250px:
$("button").click(function(){
$("div").animate({left: '250px'}); });

jQuery animate () - Utiliza la funcionalidad de cola

Por defecto, jQuery viene con funcionalidad de cola para animaciones.
Esto significa que si escribe varias animate()llamadas una tras otra, jQuery crea una cola "interna" con estas llamadas de método.
Luego ejecuta las llamadas animadas UNO por UNO.
